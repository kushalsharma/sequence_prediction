After we model our data and estimate the accuracy of our model on the training dataset, we need to get an idea of the skill of the model on new unseen data. For a normal classification or regression problem, we would do this using cross validation. With time series data, the sequence of values is important. A simple method that we used is to split the ordered dataset into train and test datasets. The code below calculates the index of the split point and separates the data into the training datasets with 75% of the observations that we can use to train our model, leaving the remaining 25% for testing the model.

We create a create data set function that takes two arguments: the dataset, which is a NumPy array that we want to convert into a dataset, and the look_back, which is the number of previous time steps to use as input variables to predict the next time period, in this case defaulted to 1.

This default will create a dataset where X is the price if the stock at a given time (t) and Y is the proce of the stock at the next time (t + 1).

In [112]:
def create_data_set(data_set, look_back=1):
    data_x, data_y = [], []
    for i in range(len(data_set) - look_back - 1):
        a = data_set[i:(i + look_back), 0]
        data_x.append(a)
        data_y.append(data_set[i + look_back, 0])
    return numpy.array(data_x), numpy.array(data_y)

We create another create future data set function that takes three arguments: original data set, which is the used to fill the previous X values in the future dataset, the dataset, which is a NumPy array that we want to convert into a future dataset, and the look_back, which is the number of previous time steps to use as input variables to predict the next time period, in this case also defaulted to 1.

In [113]:
def create_future_data_set(original_data_set, data_set, look_back=1):
    data_x, data_y = [], []
    a = original_data_set[-(look_back):, 0]
    data_x.append(a)
    data_y.append(0)
    return numpy.array(data_x), numpy.array(data_y)

The LSTM network expects the input data (X) to be provided with a specific array structure in the form of : [samples, time steps, features].

Currently, our data is in the form : [samples, features] and we are framing the problem as one time step for each sample. We can transform the prepared train and test input data into the expected structure using numpy.reshape()

Now we build the LSTM network. The network has a visible layer with 1 input, two hidden layer with 32 and 16 LSTM blocks or neurons each and an output layer that makes a single value prediction. The linear activation function is used for the LSTM blocks. We fit the model with training data. The network is trained for 5 epochs and a batch size of 1 is used. We used 'mean_squared_error' as the loss function and 'adagrad' as optimizer.
